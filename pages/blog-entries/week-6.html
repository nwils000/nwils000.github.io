What does it mean to develop good code?

Developing good code means three key things: #1 it has to work, #2 it has to be DRY, #3 it has to be easily readable.

What does it mean to be a good developer?

A good developer can code quickly, knows where to look and how to solve problems, and can debug effectively. Most importantly, a good developer delivers quality code that not only works but can also be updated and read by anyone.
